# 更新菜单指南

使用 `node build/generateMenuForCategory.js 文件夹名` 来进行生成 .menu.json 文件

如果 docs 下新增了文件夹，编辑 docs/.menu.json 文件

执行 `npm run generate:sidebar` 更新 config.mts 中的 sidebar

执行 `npm run gen-content` 更新 docs/content.md 文件，是文件汇总

## Makefile 快速命令

| 命令 | 说明 |
|------|------|
| `make dev` | 启动开发服务器 |
| `make build` | 构建生产版本 |
| `make preview` | 预览生产构建 |
| `make gen-menu folder_name` | 为指定文件夹生成 .menu.json |
| `make gen-root-menu` | 生成根菜单 |
| `make generate-sidebar` | 更新 config.mts 中的 sidebar |
| `make gen-content` | 更新 docs/content.md 文件 |
| `make update-menu` | 完整菜单更新流程（generate-sidebar + gen-content） |
| `make help` | 显示所有可用命令 |
