# Vue 官方 vscode 插件 Vue(Official) 停止对 Vue 2 进行支持

根据 Vue Official 的插件的 github 上的[更新日志](https://github.com/vuejs/language-tools/blob/master/CHANGELOG.md)显示,从 3.1.0 开始已经去除了 Vue 2 的支持，就是说如果我们要同时支持 Vue2 和 Vue3 的语法的话，我们最大只能安装 3.0.10 版本的 Vue Official 插件

如果说你的项目是使用 Vue2 和 Vue3 同时进行开发的，建议安装 3.0.10 版本的 Vue Official 插件
