# 项目文档工具如何选择

## VitePress

VitePress 是由 Vue 驱动的静态网站生成框架。 如果你想要一个基于 Vue 的解决方案，VitePress 是个不错的选择。

## Docusaurus

<https://docusaurus.io/docs>

和 VitePress 差不多，Docusaurus 是一个基于 React 驱动的静态网站生成框架。

## Rspress

<https://rspress.rs/>

Rspress 是一个基于 Rspack 构建的静态网站生成框架。

非常地快速

## MkDocs

MkDocs 是一个基于 Python 的静态网站生成框架。

## Jekyll

老牌的基于 Ruby 的静态网站生成框架，建议迁移到其他框架，playwright 转了 Docusaurus
