# AI 时代如何快速学习源码

github 上的项目如何学习源码？

推荐两种方式：

## DeepWiki

如果是英文阅读没有问题的话，推荐使用 github 上的 DeepWiki，如果想学习什么项目，就进入这个项目，然后把网址中的`https://github.com/`替换成`https://deepwiki.com/`，然后就可以进入 DeepWiki 阅读了

里面有比较好的解释，还有很多流程图，有 codemap, 支持问答，可以快速了解项目

## zread

官方网址：<https://zread.ai/>

是国内的团队开发的一个项目，可以帮助我们快速了解项目

支持自己添加 github 上的项目，支持中英文
