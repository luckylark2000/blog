# 解决 windows11 系统中睡眠后黑屏唤不醒的问题（亲测有效）

## 方法一

Win+X，在列表中选择设备管理器，找到：

系统设备->Intel(R) Management Engine Interface #1

右键->属性->电源管理->**取消勾选**：允许计算机关闭此设备以节约电源

## 方法二

在`控制面板\所有控制面板项\电源选项\系统设置`下使用管理员权限点击`更改当前不可用设置`

**取消勾选**：启用快速启动（推荐）

## 其他方法

在`控制面板\所有控制面板项\电源选项`中的选择或自定义电源计划中选择`高性能`，然后点击`更改计划设置`、`更改高级电源设置`，修改`硬盘->在此时间后关闭硬盘`时间要尽可能大，`100000`或者`从不`，（PS:`从不`在 windows 里面设置之后会自动改为`0`，是一个意思，就是永不自动关闭硬盘）
