# Mac 中使用openInTerminal把快捷方式添加到右键菜单

电脑使用比较常见的需求就是，把一些常用的工具添加到右键菜单，比如说 terminal终端或者 VSCode。

openInTerminal这个工具就是干这个的，是开源免费的。按照官方文档进行安装和使用就可以了。

需要注意的是，图标没有，右键菜单添加不生效的时候，去查看仓库里面的英文版的配置说明：<https://github.com/Ji4n1ng/OpenInTerminal/blob/master/Resources/README-Config.md>。主要是手动开启系统默认关闭的`Finder Extension`
