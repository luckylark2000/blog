# HTTP服务端错误状态码巧记

## 服务端错误状态码

服务端错误状态码：

- 500 服务器内部错误。服务崩了
- 501 服务不支持，未实现
- 502 连接错误，网关或者代理错误。一般是 Nginx 错误
- 503 服务器不可用，服务器无法处理请求，常见原因是过载或者在维护。比如说抢票的时候。
- 504 网关超时。接口比较慢。

## 巧记

- 500 零蛋，啥都没有。服务器坏了，啥都干不了了。
- 501 “五一劳动节”，一直劳动，干苦力，没技术。对应：未实现
- 502 “502胶水”出问题。连接不上，网关错误。
- 503 “我零三”，时间零散。腾不出来时间。服务器繁忙
- 504 “我零死” 等到死了，响应还没来。服务器超时。
